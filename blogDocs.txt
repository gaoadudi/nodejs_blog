- Chạy app:
$ npm start => Run app
$ npm run watch => Auto render scss to css

- Cấu trúc thư mục:
> nodemon: Tự reset app khi phát hiện thay đổi
+ .gitignore: Các file/folder sẽ không push lên remote repo
+ package.json: Cấu hình hệ thống
+ nodemon.json: Các kiểu file mà nodemon sẽ lắng nghe thay đổi
+ src:
1. index.js: File chính (route, ...)
2. public: css, tài nguyên (img, ...)
3. resources: Tập hợp các file code
> handlebars: Template
views/layouts/main.hbs => main là template luôn được gọi, các thành phần khác sẽ được render thêm vào
views/partials => Chứa các thành phần phụ sẽ được render vào main
> sass (scss): 1 cách khác để viết css (gọn và dễ quản lý hơn)
scss/app.scss => Sẽ tự render sang file app.css
scss/_variables.scss => Chứa các biến giá trị có thể dùng nhiều lần trong css
